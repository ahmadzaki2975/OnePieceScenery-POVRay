Program utama tersedia di file 480179-AhmadZakiAkmal-UAS-TVG.pov,
sesuai dengan ketentuan dari soal.

Semua library adalah library bawaan dari POV-Ray,
tetapi tetap disediakan jika diperlukan. Library yang digunakan adalah:
- colors.inc
- woods.inc
- golds.inc

Objek-objek yang dibuat dipisah dalam 4 file objects yang berbeda
- StrawHat.pov
- EternalPose.pov
- Map.pov
- Coins.pov

Bentuk primitive yang digunakan adalah sebagai berikut:
- Sphere
- Cone
- Cylinder
- Box
- Plane

Map.pov memerlukan asset gambar bernama Map.jpg.

Apabila ditemukan hasil pengerjaan dari mahasiswa lain yang mirip, 
bukti bahwa saya mengerjakan tugas ini secara mandiri bisa dilihat 
pada repositori GitHub berikut 
https://github.com/ahmadzaki2975/OnePieceScenery-POVRay
di sana terdapat catatan git commit dari awal pengerjaan sampai selesai.
